Sigma: your window into the data

What Sigma is

  • Zinc's BI (Business Intelligence) tool — signed 2yr contract May 2026
  • Live, interactive dashboards built on warehouse data
  • Works like a spreadsheet — no SQL required
  • Point-and-click filtering, drill-down, and pivot
  • Export to CSV from any table

How to access

  • Go to app.sigmacomputing.com
  • Sign in with Google (your @zincwork.com email)
  • Ask Lex if you don't have access yet

What Sigma is not

  • ❌ Not a replacement for Google Sheets (it's read-only on live data)
  • ❌ Not a place to edit or update records
  • ❌ Not a replacement for HubSpot for CRM activity
  • ✅ A place to explore, analyse, and understand what's happening in the business

Finding your way around

Home screen

  • Recent workbooks show automatically
  • Star any workbook to add it to Favourites
  • Use the search bar (top) to find workbooks by name
  • Browse by folder to find team content

Inside a workbook

  • Page tabs at the bottom — like sheet tabs in Sheets
  • Filter controls at the top of each page
  • Click any chart element to cross-filter the page
  • Hover over data points to see exact values

Exporting data

  • Click the three-dot menu (⋮) on any table
  • Select Export → CSV
  • Applies your current filters — what you see is what you get
  • Max ~1M rows per export
Quick tips for filtering
📅 Date range — most pages have a global date filter at the top. Always set this first.
🏢 Company filter — filter to a single customer to answer account-specific questions.
✅ Check type — filter to a specific check type (DBS, ID, Reference, etc.) to isolate that product.

Getting help and requesting data

💬
Got a question?
Post in Slack:
#data-requests

Describe what you're trying to answer — not the SQL you want. Lex + Zhen triage daily and will either point you to an existing workbook or help you build the query.
🐛
Found a discrepancy?
If numbers don't match what you expect, log it in #data-requests with:

What you were looking at
What you expected to see
What you actually saw

Include a screenshot if you can.
🏗️
Want something new built?
Raise a GSD project request. New Sigma dashboards and warehouse views are scoped and prioritised through the standard GSD process.

The clearer your "what question am I trying to answer?" the faster it gets built.
