What it is

Sigma
  • Zinc's BI tool — live data, interactive dashboards
  • Sits on top of the data warehouse (all Zinc systems in one place)
  • No SQL — point-and-click filtering, drill-down, pivot
  • Works like a spreadsheet but runs against live data
  • Export any table to CSV
How to get in

  • HubSpot → deals, pipeline, interactions
  • Zendesk → support tickets (coming Q3)
  • Google Analytics + Search Console → web traffic
  • All joined, cleaned, and documented

How to navigate it

  • 1
    Home screen
    Recent and favourited workbooks. Use the search bar to find anything by name. Star a workbook to keep it in your favourites.
  • 2
    Inside a workbook
    Page tabs at the bottom (like sheets in Sheets). Filter controls at the top — set date range first, then company or check type. You can't break anything; your session is local.
  • 3
    Exploring
    Click any chart element to cross-filter the page. Hover for exact values. Click column headers to sort. Right-click a cell to filter to that value or drill down.
  • 4
    Building your own view
    Hit + to add a new page tab. Drag in columns, apply grouping and filters. Use this to explore questions not covered by an existing page.
  • 5
    Exporting
    Three-dot menu (⋮) on any table → Export → Download as CSV. Exports your current filtered view exactly as shown.
